DCI Top 5 by Thomas Adamson

Welcome to DCI Top 5! This website is a collection of Drum Corps Intenational World Class Scores from 2013 to 2019.

Drum Corps International (DCI) is an organization of competitive Drum Corps worldwide. Each summer, high school and college age students tour the United State with their Corps, performing their field programs for fans. World Class Finals are held in early August, where the top 12 Corps from the summer compete for the World Class Championship.

This site shows a table and graphical representation of the last seven finals championships. For each year, you can find a line graph of each of the top five Corps' scores throughout the season. You will also find a table of scores broken down by the three judgement captions: General Effect, Visual, and Music. Caption winners are highlighted in gold. These scores were put in manually from the Official DCI Scores page. Lastly, there is a table with each Corps' show title and link to their music repertoire in DCX, an archive of past shows.

The line graphs were created using a dataset from Kaggle which contained top-line scores from each Corps at each event. You can find the original table on the "Data" page above. With some manipulation, I determined the top five scoring Corps for each season, then graphed each Corps' season by date, culminating in Finals Night.

Enjoy!
